DTO types

DTO types are business categories that were made by dividing all business tasks and functions to an understandable logic groups. DTO, business tasks and functions - have the same types.

We assume DTO types with the same way as Functions and Tasks to simplify their interconnection and relationship understanding. Types in turns may be divided to the subtypes.

DTO for admin DBOS functions
serves to fulfil admin-related functions within digital banking (DB) solution.
DTO for security functions
serves to fulfil security functions within DB solution.
DTO for business functions
serves to fulfil all functions that have to do with revenue generation.
DTO for communication functions
serves to fulfil two-way (chatbot, email etc.) and one-way (notifications) t communication functions.
DTO for report and analysis functions
serves to fulfil functions that have to do with data analysis and presenting them in a form of different reports.
DTO for management functions
serves to fulfil functions that are within managers (C-level, managers, team leads etc.) agenda.
DTO for DBOS maintenance functions
serves to fulfil functions that have to do with software maintenance operations like: version updating, scaling, running backups, performing health checks.
